# PyQt6-NotePad
Simple NotePad that written in PyQt6
# Execute Program
## How to run?
```bash
# Clone the repo :
git clone https://github.com/eleguns/PyQt6-NotePad.git
# Go to path
cd PyQt6-NotePad
# Installing requirements :
pip install -r requirements.txt
# Run the main file
python3 NotePad.py
```
